# AGI

Expert system for building, testing, evaluating, and improving intelligent AI systems.
Uses Poseidon and PAI architectures as real-world case studies.

## Workflow Routing

| Request Pattern | Route To |
|---|---|
| Audit AI system, evaluate architecture, assess quality | `workflows/audit.md` |
| Simulate AI, test system, create test project, observe flow | `workflows/simulate.md` |
| AI security, safety assessment, prompt injection | `workflows/security-assess.md` |
| Classify task, complexity scoring, effort classification | `workflows/classify.md` |
| Compare AI systems, gap analysis, what does X do better | `workflows/compare.md` |
| Optimize thinking, inference optimization, bottleneck | `workflows/optimize.md` |
| Research methodology, which research source, single vs multi-agent | `workflows/research-strategy.md` |

## Examples

**Example 1: Audit an AI system**
```
User: "Audit the Poseidon system architecture for gaps"
→ Routes to audit workflow
→ Evaluates against founding principles + self-healing/learning criteria
→ Returns structured report: strengths, gaps, recommendations
```

**Example 2: Simulate and test**
```
User: "Simulate creating a complex project and observe the AI flow"
→ Routes to simulate workflow
→ Creates test projects of varying complexity
→ Measures thinking process, rule adherence, efficiency, quality
→ Returns evaluation report with scores
```

**Example 3: Research methodology guidance**
```
User: "Should I use single or multi-agent research for this?"
→ Routes to research-strategy workflow
→ Analyzes question complexity and type
→ Recommends source selection and depth
```
